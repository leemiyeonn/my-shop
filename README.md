# 🛍️ my-shop

사용자가 쉽게 상품을 검색하고 관리할 수 있는 간단한 쇼핑 애플리케이션입니다.

## 🛒 배포 정보

http://43.203.253.0:8080/

## 🛒 테스트 계정

- 아이디: test1
- 비밀번호: 0000

## 🛒 주요 기능

### 🛍️ 1. 상품 검색
- 네이버 쇼핑 API와 연동하여 다양한 상품을 검색할 수 있습니다.
- 키워드 기반 검색 지원

### 🛍️ 2. 최저가 설정
- 사용자가 원하는 상품의 최저가를 설정할 수 있습니다.
- 최저가 알림 기능 제공

### 🛍️ 3. 폴더에 담기
- 관심 상품을 개인 폴더에 저장할 수 있습니다.
- 폴더 생성 기능 제공

## 🛒 기술 스택

- Spring Boot
- Thymeleaf
- AWS
- RDBMS (MySQL)
